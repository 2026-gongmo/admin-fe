- [ ] 사이드바 메뉴가 정상 이동되는가?
- [ ] 주요 표의 텍스트가 잘리지 않는가?
- [ ] 상세 패널에서 버튼/입력창이 겹치지 않는가?
- [ ] 검색 결과 없음 상태가 표시되는가?
- [ ] Mock 기능은 실제 저장처럼 표현하지 않는가?
- [ ] `백엔드 붙여야 함`, `Mock`, `추가 필요` 표현이 필요한 곳에 보이는가?
- [ ] PDF/CSV 내보내기는 Mock Toast로만 동작하는가?
- [ ] 위험 작업 전 ConfirmModal이 표시되는가?
- [ ] 로딩/빈 상태가 공통 PageState 형태로 표시되는가?
- [ ] 목록 로딩 중 SkeletonTable이 깨지지 않는가?
- [ ] 운영 동기화 상태 바가 Mock 상태를 명확히 보여주는가?
- [ ] 제보/도움 요청/워크플로우 상세 패널 닫기와 다시 열기가 동작하는가?
- [ ] 제보/도움 요청/워크플로우 상세 패널 고정/해제 버튼 문구가 명확한가?
- [ ] 제보/도움 요청/워크플로우 상세 패널의 이전/다음 버튼과 `현재 / 전체` 표시가 맞는가?
- [ ] 제보/도움 요청/경험 피드/워크플로우 필터 초기화 버튼이 검색어와 필터를 정상 초기화하는가?
- [ ] 도움 요청과 워크플로우 정렬 기준/방향을 바꿔도 표가 깨지지 않는가?
- [ ] 상단 권한 미리보기에서 센터/시설팀/슈퍼관리자 전환 시 메뉴 노출이 의도와 맞는가?
- [ ] 발표 모드 토글 시 개발 안내 문구 일부가 숨겨지고, Mock 데이터 상태는 오해 없이 남아 있는가?
- [ ] 콘솔 에러가 없는가?

## 추천 검색 명령

```bash
rg -n "API_KEY|SECRET|TOKEN|PASSWORD|BEGIN PRIVATE|sk-|ghp_|xoxb-|DB_PASSWORD|\\.env" . \
  --glob '!node_modules/**' \
  --glob '!dist/**' \
  --glob '!*.tsbuildinfo' \
  --glob '!.git/**'
```

검색 결과가 나오면 실제 비밀값인지, 문서상의 주의 문구인지 구분해서 확인합니다.
